# Malaysia Flood Risk AI

A Malaysia-focused flood risk prediction and explainability project.

## Objective

This project estimates flood risk for locations in Malaysia using geospatial, weather, hydrological, and machine learning features.

## Planned Features

- Malaysia-wide flood risk scoring
- Weather forecast and warning integration
- Terrain, slope, land cover, and river distance features
- Explainable machine learning model
- FastAPI prediction API
- Streamlit public demo application
- Clean research notebooks
- Reproducible data and model pipeline

## Tech Stack

- Python 3.12
- GeoPandas
- Rasterio
- scikit-learn
- XGBoost
- LightGBM
- FastAPI
- Streamlit
- MLflow
- DVC

## Disclaimer

This project is for research, education, and public awareness. It should not be used as the sole basis for emergency decisions.
